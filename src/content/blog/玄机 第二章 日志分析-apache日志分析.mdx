查看黑客IP访问了多少次，黑客 IP 是 `192.168.200.2`，配置 `wc -l` 查看一下

```shell
grep -a "192.168.200.2 - - " access.log.1 | wc -l
```

![](https://img.trtyr.top/images/blog/%E7%8E%84%E6%9C%BA%20%E7%AC%AC%E4%BA%8C%E7%AB%A0%20%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90%20Apache%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90/003%20%E7%AC%AC%E4%BA%8C%E7%AB%A0%20%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90-apache%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90-3.png?imageSlim)

查看2023年8月03日8时这一个小时内有多少IP访问，直接筛选时间

```shell
grep -a "03/Aug/2023:08:" access.log.1
```

然后提取 IP，一般来说每行开头就是 IP，我们使用 `awk`

```shell
grep -a "03/Aug/2023:08:" access.log.1 | awk '{print $1}'
```

接着使用 `sort -nr` 排序

```shell
grep -a "03/Aug/2023:08:" access.log.1 | awk '{print $1}' | sort -nr
```

然后使用 `uniq -c` 命令统计一下

```shell
grep -a "03/Aug/2023:08:" access.log.1 | awk '{print $1}' | sort -nr | uniq -c
```
