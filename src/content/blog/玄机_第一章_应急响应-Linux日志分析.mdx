## WP

首先是查看有多少 IP 在爆破 SSH，我们看日志，位置在 `/var/log`

![](https://img.trtyr.top/images/blog/%E7%8E%84%E6%9C%BA%20%E7%AC%AC%E4%B8%80%E7%AB%A0%20%E5%BA%94%E6%80%A5%E5%93%8D%E5%BA%94-Linux%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90/002%20%E7%AC%AC%E4%B8%80%E7%AB%A0%20%E5%BA%94%E6%80%A5%E5%93%8D%E5%BA%94-Linux%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90-1.png)

我们主要看 `auth.log` 和 `auth.log.1`，可能是在备份文件 `auth.log.1` 里。

SSH 登录爆破，会有关键字 `Failed password` 和 `Accepted` 出现，我们进行筛选

```shell
grep -a "Failed password" auth.log.1
```

![](https://img.trtyr.top/images/blog/%E7%8E%84%E6%9C%BA%20%E7%AC%AC%E4%B8%80%E7%AB%A0%20%E5%BA%94%E6%80%A5%E5%93%8D%E5%BA%94-Linux%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90/002%20%E7%AC%AC%E4%B8%80%E7%AB%A0%20%E5%BA%94%E6%80%A5%E5%93%8D%E5%BA%94-Linux%E6%97%A5%E5%BF%97%E5%88%86%E6%9E%90-2.png)

得到 IP 如下

```
192.168.200.35
192.168.200.32
192.168.200.2
192.168.200.31
```

提交，失败，重新看题，发现要求的是爆破 root 用户的 IP，没有 `192.168.200.35`，重新提交，通过。

成功登录的 IP，很明显就是 `192.168.200.2`。

爆破用户名字典是什么？直接看图就能看出来，`test1,test2,test3,root,user,hello`

登陆成功的 IP 共爆破了多少次，数数，`192.168.200.2` 对 root 用户爆破了 4 次

黑客登陆主机后新建了一个后门用户，我们找命令 `useradd`

```shell
grep -a "useradd" auth.log.1
```
